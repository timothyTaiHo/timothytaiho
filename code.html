Front End:

Front End Web Development is basically the development of how the site looks.

Back End:

Back End Web Development is how the site works. It has to do with databases and servers, which has to do with how the site works.

HTML:

HTML is the basic structure of a website, like the words.

CSS:

CSS has to do with the styling of the website, like the colors, fonts, the size, and buttons.

JavaScript:

Javascript is the interaction with HTML, it makes interactive effects in the website.
